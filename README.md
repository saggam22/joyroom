## ⚙프로젝트 개발환경
+ 운영체제 : Window
+ 통합개발환경(IDE) : Eclips
+ JDK 버전 : JDK 11
+ 데이터 베이스 : Oracle Database 11g
+ 빌드 툴 : Maven
+ 관리 툴 : Git, GitHub
